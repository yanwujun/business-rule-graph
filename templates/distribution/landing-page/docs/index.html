Four guides

Pick where you are. Each guide stands alone; together they take you from pip install to a CI pipeline that gates on structural review.

1. Getting Started Install, index your repo, ask the first structural question, run a pre-merge check, plan parallel agent work, sign a Code Graph Attestation. ~10 minutes start to finish. 2. Integration Tutorials End-to-end MCP setup for Claude Code, Cursor, Gemini CLI, Codex CLI, and Amp (Sourcegraph). Per-platform commands and the cross-platform validation checklist. 3. Command Reference All 205 commands, grouped by intent: exploration, health, refactoring, security, governance. Each entry has flags, output, and a concrete usage example. 4. Architecture How the index pipeline, storage, graph engines, retrieval, and patch verifier fit together. The system diagram + subsystem responsibilities + command-to-data flow.
